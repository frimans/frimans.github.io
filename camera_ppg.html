Camera PPG

In this little project I made an android app,
which tries to measure PPG (Photopletysmograp) from
users finger tip. Current version has additional 2 modes.
It can measure HR also with a seismocardiogram principle
by sesning vibrations of user's sternum. The third method is
remote PPG, in which HR is measured from user's face with the
phone's front camera.



Test it yourself! The icon below opens a drive folder, from which the APK file
can be loaded from.
